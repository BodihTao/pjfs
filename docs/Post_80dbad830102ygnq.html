地藏菩萨说信愿具足念佛必生净土与再来人不堕落秘密及此生归宿


 

作者：普敬法师

 

极乐世界再来人不堕落的秘密：


很多人都知道信愿具足的念佛可以往生极乐世界，从而了生脱死，但不知道很多极乐世界的人又发愿来到人间了，并且由于果位没达到八地以上，于是就有了入胎之谜，所以他们并不能知道自己，起码暂时是这样（除非用占察轮的观所疑设问来反复多次占察确认，或者被天眼观到），而别人也不知道他们，因为表面上看就是个凡人。

 

来人间干嘛呢？除了走捷径下来再次投生为人而念佛来求高品位往生以外，另一方面也是发愿来此世间给佛法做个陪衬，显示自己也是学佛人，以此来度化有缘众生，或者示现为大成就者来度化众生。

 

那么这种有了隔阴之谜的极乐世界再来人来了以后如果不好好修行，会不会重新迷失而堕落轮回呢？这个绝对不会的，因为阿弥陀佛十八愿里面有一大愿就是说，往生极乐世界后并发愿到他方国土度化众生者，如果还重新堕落轮回，佛就不成正觉。

 

那么最近我们的占察更是确认了这一点，地藏菩萨证明了阿弥陀佛的愿力有多伟大。

 

 

这是我一位弟子的弟子之占察结果：

 

设问：今公元2018年12月29日，农历十一月二十三日，请问地藏菩萨摩诃萨，弟子怀疑在西方极乐世界已经证得圆教初地以下果位的所有人，包括极乐世界的天人，如果其中有人，乘愿再来五浊恶世的地球做人类，假设这个人如果遇到诸多恶缘，不精进修行学佛，不守戒律，造诸恶业，总之不管修的如何，或者不管这个假设是否成立，我认为此乘愿再来的极乐世界人当生临终时，一定会再次往生到西方极乐世界。

 

绝不会在地球上连续投胎为人，绝不会在地球死时也不知道自己曾在西方极乐世界证得过的境界，除非从西方极乐世界再次发愿来人间。此人才能在地球上再次投胎为人。恭请南无地藏菩萨摩诃萨以观所疑轮相，八十三者（观所疑即为实），八十四者（观所疑为不实），为弟子慈悲护佑加持开示。

 

（全轨）第一回合，第25把，出83.观所疑即为实（怀疑想法是正确的）

 

详细数字如下：

 

第一掷（2+4+8+10+18）＝42

第二掷（1+6）＝7

第三掷（9+10+15）＝34

 

42+7+34＝总数83观所疑即为实

 

占察要点提示：占察必须做到经文提及的“数与意合“。所谓“数与意合“，就是不能答非所问，就是不能随便滚出来个数字就联想非非，必须心里想的事情能对应轮相的回答，观所疑设问而对应的答案就是观所疑83，84轮相，记住了。

 

关于这个83与84轮相的占察准不准的问题，我在上篇“占察要点”提到了（点击蓝字阅读），我公众号历史文章发布过的“占察轮教学文章”里面也有说明（点击蓝字阅读），其它数字轮相如果丢上25把，过程若没有感应地藏菩萨，那有很大蒙中的可能，而83与84这组特殊数字是很难蒙中的，除了文章提及的计算结果，以及多次试验不求菩萨来蒙这组数字，那都是蒙不中的，84偶尔还能像彩票一样被蒙中一下，83从来就没蒙到过，只有虔诚求菩萨的时候才能十几把或者二三十把就出来这种答案，这就可以保证轮相的准确性。

 

所以我们很信任这组数字，也只用这组数字问一切观所疑之事，很少用其它轮相。

 

继续正题，阿弥陀佛真是伟大，极乐世界的天人都有此等同那些有果位的圣者一样的不落轮回之待遇，那别说更高果位的极乐世界再来人了，难怪极乐世界的天人敢发愿来人间，有阿弥陀佛的愿力担保，那根本就不怕会再次堕落轮回。

 

 

而极乐世界的天人都有此待遇（注：极乐世界有天人吗？请看观无量寿经，下品往生者就是天人果位），那么他方国土的圣者是否有此能力呢？比如非极乐世界的回小向大之阿罗汉发愿来人间并且以胎生形式度众生，那么暂时有隔阴之谜是肯定的，但会不会一直迷上好几辈子呢？

 

 首先这个按照教理来说是绝对不会的，如果会的话，那初果圣人也不可能来人间七次还能证阿罗汉果了，何况阿罗汉呢？阿罗汉是位不退的圣者，即使以胎生形式度人也绝无可能迷失退位为凡夫，最迟也会在临终或者色身报尽之后恢复阿罗汉本体。

 

关于隔阴之谜的问题，上次引用瑜伽师地论说过了，除了佛和八地以上之菩萨，那么以下的果位，选择入胎度化众生的话，那在娘胎里通通避免不了隔阴之谜的。

 

至于说阿罗汉会不会来人间胎生形式度众生，这个以前文章也有讨论过了，佛在涅槃经说化身无父母无家族，无碎身舍利，化身只能搞个短暂的感应或者托梦，是故佛来人间行驶一番度化众生的事业则必须胎生，那么菩萨和阿罗汉自然也不例外。


 

而关于阿罗汉是否会因为隔阴之迷而退转的问题，那么今天我亲自用占察轮问过地藏菩萨这个问题了：

 

弟子怀疑阿罗汉乘愿再来而以胎生形式度人者（包括非极乐世界再来的阿罗汉），即使有隔阴之谜，无论修的怎样，最迟会在临终或者此生报尽之后恢复阿罗汉的本体，忆起自己是阿罗汉的记忆，绝不会继续随着隔阴之谜投胎为人，除非是恢复阿罗汉本体之后又乘愿再来人间。

 

请地藏菩萨观所疑83（观所疑即为实与）84（观所疑为不实）开示弟子。

 

20多把出83观所疑即为实（怀疑想法是正确的）

 

 

地藏菩萨说信愿具足念佛必生净土：



那么下面的事情之提问则和大家有关，也是为大家提问的，虽然阿弥陀佛第十八愿承诺只要具足信愿，就是不怀疑，具足信心，愿意去极乐世界，哪怕只有临终十念阿弥陀佛，则阿弥陀佛必来接引众生往生阿弥陀佛的国土，但是很多不是很相信，并且由于一些人违背经典的把往生之事宣扬的极难，这更让人徒生疑惑。实际上，往生极乐世界比下辈子做人还容易，因为做人需要一定的善业，而去极乐世界则完全是带业往生，只需要你信愿具足的念阿弥陀佛就可以。

 

占察如下：

 

 

案例1、

 

亲自占察：弟子认为一切众生，只要具足信愿，平生不管念诵佛号有多少，若他能够临终提起正念十念阿弥陀佛，包括默念，都能蒙承阿弥陀佛愿力成功往生极乐世界，请地藏菩萨以观所疑轮相开示弟子：

 

几十把出83观所疑即为实。（想的正确）

 

 

案例2、

 

我母亲有一次的占察也摘录给大家，她由于看了某些不如法的开示，就害怕自己不能往生，我怎么劝都不信，于是我教她占察，我说你自己问地藏菩萨，教她对菩萨的设问也和我上面所描述的差不多，她倒是很虔诚念了一千声地藏菩萨圣号而开始占察。

 

结果才几把的掷轮就得到了地藏菩萨确定的回复，但是她还不信，认为是蒙中的，于是我请她不求菩萨而去蒙蒙看能不能蒙中，结果她蒙了一上午出不来，这才服气，才坚信自己只要具足信愿，哪怕临终十念，必可成功往生。

 

 

 这个案例也可以给大家一个信心，因为她也可以代表所有的普通人，阿弥陀佛大愿是针对一切人，只要具足信愿行，这个念佛之行可以说是长在一生，短在仅仅临终的十念佛号，就都能往生的。

 

 

看到这里，大家应该放心了，对于往生极乐世界要生起绝对的信心，此生必去极乐世界，而且最好求上品上生，只要信愿具足并肯念佛，绝对没问题。

 

由于很多人暂时有很多法在修，并没有马上念诵阿弥陀佛圣号，这也是可以理解的，包括我自己也是，但没关系，知道自己的归宿是极乐世界就行，因缘到了，或者说准备好往生了，就开始拼命专一念佛求往生净土吧。


当然有一点我要说明，一般人不能抱着自己老了才念佛或者临终才念佛的想法，对于持咒者来说可以老了才念佛，因为现在持咒的功德回向往生极乐也等于在修净土，而那些平常根本就没有功课的人，光等老了或者临终，那就有老年痴呆或者临终昏昧等等的不确定性存在，最好平常多少也念念，这样老了或者临终才有定力提起正念来念佛。



阿弥陀佛第十八愿：若我法藏成佛之时，十方世界之任何众生，只要真心信受、欢喜我的救度，欲生我国，而长在一生、短在十声一声，称念我名，必使其往生，若不得往生，我誓不成佛。
